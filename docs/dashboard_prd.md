## 1. 背景与目标

### 1.1 项目背景

本系统用于创新药投资阶段评价，分三个阶段：
- 第一阶段：预期重定价（1-3月）
- 第二阶段：产业验证（4-8月）
- 第三阶段：主线强化（9-12月）

当前实现第一阶段指标计算和可视化展示。

### 1.2 界面目标

帮助用户回答以下核心问题：
1. 当前是否值得参与？
2. 标的本身表现如何？
3. 龙头和成分股表现如何？
4. 修复是否扩散到更多个股？
5. 各指标均衡性如何？

## 2. 系统架构

### 2.1 数据流

```
citydata代理 → CSV文件 → 指标计算 → JSON结果 → Streamlit界面
```

### 2.2 数据文件

| 文件 | 内容 | 用途 |
|------|------|------|
| fund_daily.csv | ETF日线行情 | 价格、成交额 |

### 2.3 技术栈

- Python 3.10+
- Streamlit（界面框架）
- Plotly（图表库）

## 4. 模块详细设计

### 4.1 综合概览

**定位**：回答"当前是否值得参与"

**展示内容**：

| 区域 | 内容 | 设计意图 |
|------|------|----------|
| 顶部状态栏 | 日期、综合得分、预期等级、数据天数 | 一眼看出结论 |
| 核心指标卡片 | S1-01、S1-02、S1-03 三个指标的值和预期等级（带颜色背景） | 抓主要矛盾 |
| 确认指标 | S1-04 成交放大持续性 | 验证修复是否持续放量 |
| 支撑指标 | S1-05、S1-06 的值和预期等级 | 辅助验证 |
| 得分趋势图 | 近20日得分折线 + 阈值线（0.80/0.60/0.40） | 判断趋势方向 |

**颜色规则**：
- 超预期：绿色背景
- 符合预期：黄色背景
- 低于预期：红色背景

**关键文案**：
- 核心指标区："判断是否'连续回来'，不是单日脉冲"
- 确认指标区："验证修复是否持续放量"
- 支撑指标区："验证是否扩散到更多个股"

---

### 4.2 ETF行情

**定位**：回答"标的本身表现如何"

**展示内容**：

| 区域 | 内容 | 对应指标 |
|------|------|----------|
| 关键数据卡片 | 最新价、涨跌幅、成交额、份额 | 快速读取 |
| 成交放大倍数 | 5日均/20日均比值，判断放量/缩量 | S1-04 |
| 价格走势图 | 589720收盘价折线 + 159557基准对比（归一化） | S1-03 |
| 成交额柱状图 | 近20日柱状图 + 5日均线 | S1-04 |
| 份额变化图 | 近期份额趋势 | S1-01/S1-02 |

**判断逻辑**：
- 成交放大倍数 ≥ 1.5x → 绿色"持续放量 ✓"
- 成交放大倍数 ≥ 1.1x → 蓝色"温和放量"
- 成交放大倍数 < 1.1x → 黄色"缩量"

---

### 4.3 成分股详情

**定位**：回答"龙头和成分股表现如何"

**展示内容**：

| 区域 | 内容 | 对应指标 |
|------|------|----------|
| 龙头组合表现 | 龙头组合涨跌 vs ETF涨跌，计算先行强度 | S1-06 |
| 龙头卡片 | 前5大持仓：代码、价格、涨跌、权重 | 快速查看 |
| 全部持仓表格 | 所有成分股：代码、权重、价格、涨跌幅 | 详细数据 |

**判断逻辑**：
- 龙头先行强度 > 0 → 绿色"龙头强于ETF ✓"
- 龙头先行强度 ≤ 0 → 黄色"龙头弱于ETF"

---

### 4.4 广度指标

**定位**：回答"修复是否扩散到更多个股"

**展示内容**：

| 区域 | 内容 | 对应指标 |
|------|------|----------|
| 涨跌分布 | 上涨/下跌/平盘数量 + 饼图 | 市场情绪 |
| 均线突破比例 | 站上20日均线的比例 + 进度条 | S1-05 |
| 均线突破明细 | 每只股：代码、价格、涨跌、MA20、状态 | 详细数据 |
| 个股排名 | 涨幅前10、跌幅前10榜单 | 找强势/弱势股 |

**判断逻辑**：
- 站上比例 ≥ 60% → 绿色"超预期 (≥60%)"
- 站上比例 ≥ 35% → 蓝色"符合预期 (35%-60%)"
- 站上比例 < 35% → 黄色"低于预期 (<35%)"

---

### 4.5 指标雷达

**定位**：综合评估6项指标的均衡性

**展示内容**：

| 区域 | 内容 |
|------|------|
| 六边形雷达图 | 6个指标值，越饱满越强 |
| 指标详情表 | 数值、预期等级、说明 |
| 权重说明 | 各指标权重占比 |

**雷达图归一化**：
- S1-04（成交放大）：比值/2.0，上限1.0
- 其他指标：直接使用，限制在0-1

## 5. 指标定义

### 5.1 第一阶段指标（6项）

| 编号 | 名称 | 角色 | 权重 | 量化口径 |
|------|------|------|------|----------|
| S1-01 | 资金回流连续性 | 核心 | 22% | 近10日份额增加天数占比 |
| S1-02 | ETF份额变化 | 核心 | 18% | 近10日份额变化率 |
| S1-03 | ETF相对强度 | 核心 | 20% | 589720收益 - 159557收益 |
| S1-04 | 成交放大持续性 | 确认 | 14% | 5日均成交额/20日均成交额 |
| S1-05 | 板块广度修复 | 支撑 | 14% | 成分股站上20日均线占比 |
| S1-06 | 龙头先行强度 | 支撑 | 12% | 龙头组合收益 - ETF收益 |

### 5.2 预期阈值

| 指标 | 超预期 | 符合预期 | 低于预期 |
|------|--------|----------|----------|
| S1-01 | ≥70% | 40%-70% | <40% |
| S1-02 | ≥3% | 0%-3% | <0% |
| S1-03 | ≥5% | 0%-5% | <0% |
| S1-04 | ≥1.5x | 1.1x-1.5x | <1.1x |
| S1-05 | ≥60% | 35%-60% | <35% |
| S1-06 | ≥5% | 0%-5% | <0% |
