# S2 Watchlist

> 本文件记录 S2 智能体观察线索，不进入 S2 正式评分。
> 正式评分只读取 `s2/data/*.csv` 中已验证、达到阈值的事件。

## 状态定义

| 状态 | 含义 | 处理 |
| --- | --- | --- |
| `watchlist` | 值得继续跟踪，但未达到正式入库阈值 | 每日复核，有新证据再升级 |
| `needs_verify` | 有线索，但来源不足或缺少交叉验证 | 明确下一步验证路径 |
| `rejected` | 已排除，不再跟踪 | 写明排除原因 |
| `promoted` | 已升级写入正式事件库 | 写明入库文件和去重依据 |
| `expired` | 超过观察期，无后续进展 | 移入历史关闭线索 |

## 使用边界

- 本文件允许记录弱线索、研究假设和待验证事项。
- 本文件不得作为 S2 正式评分输入。
- 正式事件必须先升级写入 `s2/data/*.csv`，才能进入 S2 评分。
- 每日 agent run 必须说明 watchlist 的新增、延续、升级、删除或无变化。
